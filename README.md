# MangaLeaf WordPress Theme

A modern WordPress theme optimized for manga, manhua, and manhwa content management.

## Requirements

### PHP Version
- **PHP 8.1 - 8.4** (Compatible)

### WordPress
- **WordPress 5.9+**

### PHP Extensions (Required)
- **Ioncube Loader** - For secure code execution
- **cURL** - For remote HTTP requests
- **fileinfo** - For file type detection

### Recommended Server Configuration
- PHP 8.3 or higher (Latest stable version)
- WordPress 6.0+
- MySQL 8.0+ or MariaDB 10.5+

## Installation

1. Download the MangaLeaf theme
2. Upload to `/wp-content/themes/` directory
3. Activate the theme in WordPress admin dashboard
4. Configure theme settings as needed

## Features

- Responsive design for all devices
- Support for multiple content types (Manga, Manhua, Manhwa, Blog)
- Advanced search and filtering capabilities
- Optimized performance
- Dark mode support
- Modern UI with FontAwesome 5 icons

## PHP 8.1-8.4 Compatibility

This theme is fully tested and compatible with PHP 8.1, 8.2, 8.3, and 8.4.

### Key Features for Modern PHP:
- Type-safe function implementations
- Nullable type declarations
- Named arguments support
- Match expressions
- Constructor property promotion

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile-first responsive design
